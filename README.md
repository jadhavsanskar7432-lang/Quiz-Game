# Quiz Game (Python)

A simple and interactive quiz game built using Python.
It allows users to answer multiple-choice questions and displays the final score.

---

## Features

* Multiple-choice questions
* Score tracking system
* Simple and user-friendly interface
* Questions loaded from a JSON file
* Displays result at the end

---

## Tech Stack

* Python
* JSON (for storing questions)
* Tkinter (if GUI)

---

## Project Structure

```
quiz-game/
│── app.py
│── questions.json
│── README.md
```

---

## How to Run

1. Install Python (if not installed)

2. Clone the repository:

```
git clone https://github.com/yourusername/quiz-game.git
```

3. Navigate to the folder:

```
cd quiz-game
```

4. Run the application:

```
python app.py
```
