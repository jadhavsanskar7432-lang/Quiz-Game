# Quiz Game (Python)

A simple and interactive quiz game built using Python.
It allows users to answer multiple-choice questions and displays the final score.

---

## Features

* Multiple-choice questions
* Score tracking system
* Simple and user-friendly interface
* Questions loaded from a JSON file
* Displays result at the end

---

## Tech Stack

* Python
* JSON 
* Tkinter
